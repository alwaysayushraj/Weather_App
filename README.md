# Weather App ☁️

A modern, responsive weather application that provides real-time weather updates for cities worldwide. This project features a beautiful glassmorphism UI and dynamic weather icons based on current conditions.


## ✨ Features
* **Real-time Data:** Fetches live weather information from OpenWeatherMap API.
* **Search Functionality:** Get weather details by entering any city name.
* **Dynamic Visuals:** Icons and condition descriptions change based on the weather (Rain, Clear, Clouds, etc.).
* **Responsive Design:** Styled with CSS Flexbox and animated floating elements for a premium feel.
* **Glassmorphism UI:** Modern frosted-glass effect with a vibrant gradient background.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Advanced styling, animations, and glassmorphism.
* **JavaScript (ES6+):** Asynchronous API handling (Fetch API) and DOM manipulation.
* **API:** [OpenWeatherMap API](https://openweathermap.org/)
